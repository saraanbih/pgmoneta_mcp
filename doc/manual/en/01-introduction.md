# Introduction

pgmoneta_mcp is a Model Context Protocol server for pgmoneta. It lets a client
or AI assistant talk to pgmoneta using natural language or explicit tool calls.

## What this manual covers

- How to install and run the binaries
- How to configure pgmoneta_mcp and pgmoneta
- How to communicate with pgmoneta e.g., take backups, list them, and restore them.
- How to use the native client and inspector
- How to use the individual tool chapters for each command family

## Document flow

Start with [Installation](02-installation.md), then move to [Quick start](03-quickstart.md),
then the [Client](50-client.md) and [Inspector](51-inspector.md) chapters.
